I finally got around to implementing a blog for this website, and decided to apply some new styling in the process.

I’ve migrated, what was previously a vanilla Vue.js website, to [Nuxt.js](https://nuxtjs.org/) and used the [Nuxtent](https://nuxtent.now.sh) community module to provide a [Jekyll](https://jekyllrb.com/) like, statically generated blog.

I decided to go for a more brutalist, minimal style than the last design and I’m pretty happy with how the site is looking now. 
I chose to embrace system fonts for the typography and elected to use icons to replace my previous list of used technologies.
I also scrapped the unnecessary contact page.

The website is still hosted on Github Pages and the source code can be found in this repository.

In the near future, I plan to add a static résumé to the site navigation and I’m going to try to keep up with blog posts when something worth talking about pops up.

For now, thanks for stopping by!
